About Readable Journal

Readable Journal is designed with clarity and accessibility in mind. We follow readable font sizes, clear color contrast, and keyboard-first navigation.

Design & Accessibility Goals

  • Readable base text (≥ 16px; configured at 18px).
  • Line-height set to at least 1.5 for better reading comfort.
  • All images contain descriptive alt text including the phrase "Readable Journal".
  • Navigation and forms usable by keyboard only.

Self-Audit Notes (high level)

  • H1 used and contains keyword "Readable Journal".
  • Headings organized: H1, multiple H2/H3 sections applied.
  • Base font-size 18px, line-height 1.5. Large headings ≥ 24px.
  • Alt text and filenames include keyword (placeholder images used).
  • Contrast and focus styles provided. Links have clear anchor text.
  • At least 3 external links included. Footer links and social icon included.
  • Form present with client-side validation and success message (local demo).
  • Images, audio, and video include captions/transcripts and controls.
  • Navigation intuitive and keyboard accessible; skip link included.
  • Single external CSS file `styles.css` controls styling across the site.
